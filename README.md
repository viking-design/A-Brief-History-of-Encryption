# A-Brief-History-of-Encryption

## ❗❗❗This project is currently a work in progress❗❗❗

## ℹ️ Overview
This is a project inspired by a talk I gave at my university's ethical hacking society about the brief history of encryption. This program's main aim is to educate and demonstrate how the different encryption methods in my talk function and their histories.

## 🌟 What this program contains
This program contains the following encryption methods:
* Scytale
* Caesar Cipher
* Vigenère Cipher
* Hebern Cipher/Engima Machines
* RSA Encryption
